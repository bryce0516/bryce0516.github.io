# docker-doc


# docker image command
```
// 도커 이미지 정보 
docker image inspect <imageName>


// 도커 이미지 태그정보 변경
docker image tag <imageName> <wantedName like dole/nginx>

// 태그정보 변경된 이름으로 푸쉬
docker image push <targetName>

// 도커 로그인/로그아웃
docker login/logout

// 로그인 정보확인
cat .docker/config.json
```
